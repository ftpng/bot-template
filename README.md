# Python Discord Bot Template

A simple and clean starter template for Python Discord bots.  


##  Getting Started

Follow these steps to copy this template into your own project folder.

### 1. Clone the repository
```bash
git clone https://github.com/ftpng/bot-template.git
```

### 2. Enter the project folder
```bash
cd bot-template
```

### 3. Install required packages
```bash
pip install -r requirements.txt
```

### 4. Environment Setup

Create a .env file in the root directory and add the following variables:

```
TOKEN = 

DBUSER = 
DBPASS = 
DBNAME = 
DBENDPOINT = 
```

These are used for bot authentication and database connectivity.
